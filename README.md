# Agent Skills Repository

本仓库用于存放和管理自用的 AI Agent 技能（Skills）。这些技能旨在提升开发效率，涵盖了自动化编码、CI 诊断、前端审查及打包自动化等多个场景。

## 技能列表汇总

| 技能名称 | 目录 | 核心功能 | 适用模型 |
| :--- | :--- | :--- | :--- |
| **智慧编码 (AI Coding)** | `ai-coding` | 全自动开发流程，从需求分析到 PR 合并的闭环自动化。 | 通用 |
| **Woodpecker CI 诊断** | `woodpecker-ci-triage` | 自动检查 CI 状态，失败时拉取并分析日志定位根因。 | 通用 |
| **前端 PR 审查** | `pr-review-frontend` | 针对前端 PR 的一次性深度代码质量评审。 | 通用 |
| **Argos 视觉优化** | `argos-visual-timing` | 优化 Argos/Playwright 截图时机，解决截图太早/太晚问题。 | 通用 |
| **打包脚本专家** | `build-scripts` | Android (React Native) 打包自动化，确保环境纯净与 100% 成功率。 | 通用 |

## 详细技能说明

### 🤖 智慧编码 (`ai-coding`)
- **定位**：全自动执行者，而非单纯的提醒者。
- **工作流**：状态自检 -> 环境初始化 -> 编码与测试 -> 资源清理 -> 创建 PR -> **持续监控循环**。
- **核心特色**：具备“零打扰”原则，监控循环会自动修复 CI 失败、解决代码冲突并响应 Review 反馈。

### 🪵 Woodpecker CI 诊断 (`woodpecker-ci-triage`)
- **定位**：解决“为什么 CI 挂了”的快速响应专家。
- **功能**：自动识别当前仓库与分支，智能等待流水线初始化，仅抓取失败 Step 的关键日志。
- **常见模式**：内置了针对依赖超时、编译失败、权限错误等常见故障的速查表。

### 🔍 前端 PR 审查 (`pr-review-frontend`)
- **定位**：一次性、深度的前端代码评审。
- **审查视角**：包含通用规范、JS 逻辑、项目结构、静默失败猎手、前端 Spec 检查等多个专业维度。
- **输出**：生成高质量的中文评审报告，重点突出阻塞性问题。

### 📸 Argos 视觉优化 (`argos-visual-timing`)
- **定位**：提升视觉回归测试的稳定性。
- **策略**：将截图分为“早截”（捕捉入口态）和“晚截”（等待内容锚点稳定），减少对固定 `waitForTimeout` 的依赖。
- **目标**：消除重复态截图，确保 Argos Diff 的有效性。

### 📦 打包脚本专家 (`build-scripts`)
- **定位**：严苛模式下的 Android/RN 打包自动化。
- **原则**：零污染（打包后彻底还原）、物理隔离（清理 node_modules）、全量覆写（保证环境一致性）。
- **能力**：包含 NDK 崩溃修复、MMKV 镜像优化、AndroidX 锁定等一系列黄金实践。

## 使用指南
每个技能目录下均包含一个 `SKILL.md`，其中详细说明了该技能的触发场景、环境变量要求及具体工作流。你可以根据需要将特定技能加载到你的 AI 助手（如 Gemini CLI, Claude Desktop 等）中使用。
